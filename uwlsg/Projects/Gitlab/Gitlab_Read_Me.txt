# Install Gitlab locally using Vagrant

Main page for Gitlab installation instructions is:

https://about.gitlab.com/


Vagrant install instructions are located at:

https://about.gitlab.com/installation/

https://github.com/tuminoid/gitlab-installer

Download github repository to local machine working directory

VirturalBox Version:
Version 5.1.22 r115126 (Qt5.6.2)

Vagrant version:
Vagrant 1.9.6

Review the directory contents.

Setup new host name in /etc/hosts file
-- https://gitlab.local  127.0.0.1

Change directory to working directory
-- cd  workingDirectory

Run the vagrant up command
-- vagrant up

After VM is running enter VM using the vagrant ssh command.
-- vagrant ssh
